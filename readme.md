# Typing Practice

一个纯前端打字练习项目（无账号系统），当前已完成阶段二核心能力：多训练模式、智能难度、复盘分析、反作弊检测。

## 当前能力
- 支持内置样本文本与 `.txt` 上传
- 支持 `texts/` 目录文章清单加载（启动随机加载一篇，可手动切换）
- 支持超长文本（窗口化渲染，避免全量字符 DOM 卡顿）
- 输入框与显示框固定尺寸，仅纵向滚动（不显示横向滚动条）
- 计时从输入第一个字符开始
- 支持两种正确率显示模式
- `Result Accuracy`：当前结果正确率
- `Process Accuracy`：输入过程击键正确率
- 支持中英文界面切换（自动识别浏览器语言，可手动切换并记忆）
- 训练模式支持悬停/聚焦查看说明弹窗
- 禁止在输入框粘贴/拖拽文本

## 阶段二功能
### 训练模式
- `Classic`：完整文本练习
- `Timed`：倒计时模式（到时自动结束）
- `Segment`：按分段长度切片练习
- `Blind`：输入时隐藏未输入文本
- `Error Focus`：基于历史错误自动生成专项练习文本

### 智能难度（Adaptive Difficulty）
- 根据本次 `WPM + Result Accuracy + Process Accuracy` 自动调节：
- 分段长度（`segmentLength`）
- 文本可见度（`opacity`）
- 可手动关闭自适应

### 复盘分析
- 输出错误率（Error Rate）
- Top 错误字符
- Top 错误词
- 本次训练建议（Training Suggestion）

### 反作弊检测
- 检测并提示可疑成绩，包括：
- 粘贴/拖拽尝试
- 异常高 WPM
- 极短时间高输入量
- 过高峰值 CPS（characters per second）

## 启动方式（推荐）
1. 在项目根目录打开终端：`c:\Project\typing`
2. 启动本地静态服务：`python -m http.server 5500`
3. 浏览器访问：`http://localhost:5500/index.html`
4. 停止服务：`Ctrl + C`

> 不建议直接双击 `index.html`（`file://` 方式），会导致目录文章列表无法读取。

## 快速开始
1. 按上面的方式启动并打开页面
2. 选择样本文本、目录文章或上传 `.txt`
3. 选择训练模式与参数
4. 点击 `Start Typing`
5. 练习结束后查看结果与复盘面板

## 指标定义
### WPM
- 公式：`结果正确字符数 / 5 / 用时分钟`

### Result Accuracy
- 公式：`结果正确字符数 / 当前输入总字符数`

### Process Accuracy
- 公式：`过程正确击键数 / 过程总击键数`

### Error Rate
- 公式：`结果错误字符数 / 当前输入总字符数`

## 文件结构
- `index.html`：页面结构与样式
- `app.js`：阶段二核心交互逻辑（模式引擎、统计、复盘、自适应、反作弊）
- `texts/`：目录文章清单与练习文本（`articles.json` + 多篇 `.txt`）
- `skills/typing-system-review/`：系统级代码审查 Skill（含 references 与 agents 元数据）
- `readme.md`：项目文档

## 已知限制
- 当前是纯前端实现，刷新后历史错误数据不会持久化
- 目录文章功能依赖本地静态服务（`file://` 直开会被浏览器拦截）
- 暂无后端存储、排行榜、多人对战
- 未接入自动化测试框架
